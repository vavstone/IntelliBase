# IntelliBase
Корпоративная база знаний с использованием ИИ.

# Тема
Ассистент для анализа документов.
# Описание
Cистематизирует и предоставляет инструменты поиска и анализа внутренней документации (форматы pdf, docx, ...) для сотрудников IT-компании.
